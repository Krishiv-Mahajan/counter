# Counter App

A simple interactive counter application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Increment**: Click the `+` button to increase the counter
- **Decrement**: Click the `-` button to decrease the counter (cannot go below 0)
- **Reset**: Click the `Reset` button to return the counter to 0
- **Validation**: Prevents counter from going negative with an alert message

## Project Structure

```
counter/
├── index.html      # HTML structure of the counter app
├── script.js       # JavaScript functionality for counter logic
├── style.css       # Styling for the counter app
└── readme.md       # Project documentation
```

## How to Use

1. Open `index.html` in your web browser
2. Click the buttons to control the counter:
   - **-** button: Decreases the count by 1
   - **Reset** button: Sets the count back to 0
   - **+** button: Increases the count by 1

## Technologies Used

- **HTML5**: Page structure
- **CSS3**: Styling and layout
- **JavaScript (Vanilla)**: Event handling and counter logic

## Browser Compatibility

Works on all modern browsers that support ES6 JavaScript.
